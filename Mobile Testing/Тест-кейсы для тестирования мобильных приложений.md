# Тест-кейсы для тестирования мобильного приложения

## TC-001
| Поле | Значение |
|:-----|:---------|
| **Название** | Создание задачи с валидными данными (кириллица) |
| **Модуль** | Создание задачи |
| **Приоритет** | High |
| **Предусловие** | Приложение открыто, список задач пуст или содержит менее 10 задач |
| **Шаги** | 1. В поле ввода ввести текст: "Купить молоко".<br>2. Нажать кнопку "Добавить задачу". |
| **Ожидаемый результат** | Задача "Купить молоко" появляется в списке задач. Счётчик задач увеличивается на 1. |
| **Статус** | Passed |
