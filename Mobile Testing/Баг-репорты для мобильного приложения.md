# Баг-репорты для мобильного приложения

## BUG-001
| Поле | Значение |
|:-----|:---------|
| **ID** | BUG-001 |
| **Краткое описание** | Отсутствует сообщение об ошибке при изменении задачи, содержащей только цифры |
| **Модуль** | Изменение и удаление задачи |
| **Приоритет** | Medium |
| **Серьёзность** | Major |
| **Окружение** | Samsung Galaxy S22+, Android 16 |
| **Шаги воспроизведения** | 1. Долгое нажатие на существующей задаче.<br>2. Ввести "12345".<br>3. Подтвердить изменение. |
| **Фактический результат** | Задача изменяется на "12345". Сообщение об ошибке не появляется. |
| **Ожидаемый результат** | Задача не изменяется. Появляется сообщение об ошибке о недопустимых символах. |
| **Статус** | Open |
